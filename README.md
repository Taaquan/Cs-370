# Cs-370

Pirate Intelligent Agent – Reinforcement Learning Project
Overview

This project demonstrates the development of a pirate intelligent agent using reinforcement learning and neural networks in a Jupyter Notebook. The goal of the agent is to learn how to navigate a maze and reach a treasure efficiently by learning from experience rather than following hard-coded instructions.

My Contributions

For this project, some foundational code was provided, including the maze environment (TreasureMaze), parts of the neural network setup, and helper functions.

My main contributions included:

Implementing and refining the Q-learning training loop (qtrain)
Managing experience replay memory for training the model
Adjusting key parameters like epsilon (exploration vs. exploitation)
Debugging training issues and improving model performance
Ensuring the agent learned to make better decisions over time

This work allowed the agent to transition from random movement to intelligently navigating the maze.
